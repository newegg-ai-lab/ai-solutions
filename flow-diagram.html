🪪
Identity Key
user_id = str(tg_user_id) — scopes all Mem0 vectors and history per user.
💬
Onyx Session
Unique Onyx chat session UUID per user, stored in sessions.json. RAG context fully isolated.
🗄️
Mem0 Bucket
Qdrant vectors and mem0_history.db partitioned by user_id — no cross-contamination.
⚡
Auto-Provision
First message from a new user automatically creates a session + bucket — zero admin overhead.
