AI Filmmaking Setup Guide

Fresh Hardware — generate and animate film shots entirely on your own GPU

Overview

Everything runs locally in ComfyUI — no cloud rendering, no per-frame fees. Stills are generated, posed, animated, combined, and upscaled entirely on the RTX PRO 6000, then handed off to DaVinci Resolve for the final edit.

Pipeline
  1. SDXL generates still frames
  2. ControlNet locks in poses & composition (3 types)
  3. AnimateDiff animates the shot
  4. Frames are combined into a clip
  5. Upscale & interpolate for smoothness
  6. Import into DaVinci Resolve for final edit
📁
Install location: pick any folder for ComfyUI — e.g. C:\Users\yourname or D:\AI. Throughout this guide that folder is written as <INSTALL_DIR>; substitute your actual path everywhere you see it.

00 Prerequisites

Verify before starting
Windows CMD 
nvidia-smi          # confirms GPU + driver + CUDA version
python --version    # need Python 3.10+
git --version

01 Install PyTorch with CUDA Support

Step 1
PyTorch + CUDA 12.8
Windows CMD 
python -m pip install --upgrade pip
python -m pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu128
✅
Verify it actually sees the GPU:
Windows CMD 
python -c "import torch; print(torch.cuda.is_available(), torch.cuda.get_device_name(0))"

Should print True and your GPU's name — if not, stop here and fix this before continuing.

02 Install ComfyUI

Step 2
ComfyUI
Windows CMD 
cd <INSTALL_DIR>
git clone https://github.com/comfyanonymous/ComfyUI.git
cd ComfyUI
python -m pip install -r requirements.txt
✅
Verify it boots:
Windows CMD 
python main.py --listen 127.0.0.1 --port 8188

Look for Device: cuda:0 <your GPU name> in the log, then Ctrl+C to stop it for now.

03 Download SDXL (Text-to-Image)

Step 3
SDXL base, refiner & VAE — ~13GB total
Windows CMD 
cd <INSTALL_DIR>\ComfyUI\models\checkpoints
hf download stabilityai/stable-diffusion-xl-base-1.0 sd_xl_base_1.0.safetensors --local-dir .
hf download stabilityai/stable-diffusion-xl-refiner-1.0 sd_xl_refiner_1.0.safetensors --local-dir .
cd ..\vae
hf download madebyollin/sdxl-vae-fp16-fix sdxl_vae.safetensors --local-dir .

04 Install ControlNet

Step 4
ControlNet Aux + Canny/Depth SDXL — ~5GB models + node pack
Windows CMD 
cd <INSTALL_DIR>\ComfyUI\custom_nodes
git clone https://github.com/Fannovel16/comfyui_controlnet_aux.git
python -m pip install -r comfyui_controlnet_aux\requirements.txt

cd <INSTALL_DIR>\ComfyUI\models\controlnet
hf download diffusers/controlnet-canny-sdxl-1.0 diffusion_pytorch_model.fp16.safetensors --local-dir .\canny-sdxl
hf download diffusers/controlnet-depth-sdxl-1.0 diffusion_pytorch_model.fp16.safetensors --local-dir .\depth-sdxl

05 Install AnimateDiff (Video Diffusion)

Step 5
AnimateDiff-Evolved + Video Helper Suite — ~6GB: SD1.5 base + motion module
Windows CMD 
cd <INSTALL_DIR>\ComfyUI\custom_nodes
git clone https://github.com/Kosinkadink/ComfyUI-AnimateDiff-Evolved.git
git clone https://github.com/Kosinkadink/ComfyUI-VideoHelperSuite.git
python -m pip install -r ComfyUI-AnimateDiff-Evolved\requirements.txt
python -m pip install -r ComfyUI-VideoHelperSuite\requirements.txt

cd <INSTALL_DIR>\ComfyUI\models\checkpoints
hf download runwayml/stable-diffusion-v1-5 v1-5-pruned-emaonly.safetensors --local-dir .

mkdir <INSTALL_DIR>\ComfyUI\custom_nodes\ComfyUI-AnimateDiff-Evolved\models
cd <INSTALL_DIR>\ComfyUI\custom_nodes\ComfyUI-AnimateDiff-Evolved\models
hf download guoyww/animatediff mm_sd_v15_v2.ckpt --local-dir .

06 Install Frame Interpolation & Upscaling

Step 6
ComfyUI-Frame-Interpolation + Real-ESRGAN — ~65MB
Windows CMD 
cd <INSTALL_DIR>\ComfyUI\custom_nodes
git clone https://github.com/Fannovel16/ComfyUI-Frame-Interpolation.git
# RIFE's own checkpoint (rife47.pth) auto-downloads the first time you use the node — no manual step needed

cd <INSTALL_DIR>\ComfyUI\models\upscale_models
curl -L -o RealESRGAN_x4plus.pth https://github.com/xinntao/Real-ESRGAN/releases/download/v0.1.0/RealESRGAN_x4plus.pth

Small download — these are pretrained CNNs, not diffusion models.

07 Launch & Confirm Everything Loaded

Step 7
Launch ComfyUI
Windows CMD 
cd <INSTALL_DIR>\ComfyUI
python main.py --listen 127.0.0.1 --port 8188

Check the log for a clean "Import times for custom nodes" section listing all four packs (comfyui_controlnet_aux, ComfyUI-AnimateDiff-Evolved, ComfyUI-VideoHelperSuite, ComfyUI-Frame-Interpolation) with no error lines above them, then open http://127.0.0.1:8188.

08 Validate It Actually Works

Step 8
Run the test scripts

Run the test scripts saved at <INSTALL_DIR>\ComfyUI\test_scripts\ — they exercise every piece above with a real generation, not just a file check.

📦
Total download size: ~30GB. Not included: DaVinci Resolve Studio (needs your purchased license/installer from Blackmagic — a separate manual step outside this pipeline).
